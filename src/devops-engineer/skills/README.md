# DevOps Skills

This directory contains pre-installed skill files for DevOps engineering.

## Included Skills

- **terraform-helper** - Terraform configuration assistance
- **kubernetes-assistant** - Kubernetes manifest generation

## Adding More Skills

To add more skills from the Roo Code marketplace:

1. Run `npx skills` in your workspace
2. Search for and install additional skills
3. Skills will be added to your local `.roo/skills/` directory

## Custom Skills

You can also create custom skills by adding new directories here with a `SKILL.md` file following the same format.

## Usage

These skills activate automatically based on file context when using Roo Code or Claude Code.
